# PokemonHelp

A companion web app for Pokemon FireRed & LeafGreen. Open it alongside your game for instant access to Pokedex info, location data, team building tools, and more.

## Features

- **Pokedex** -- Browse all 151 Kanto Pokemon with stats, artwork, encounters, and evolution chains
- **Location Map** -- Explore every route, city, and dungeon with available Pokemon and encounter details
- **Team Builder** -- Build a 6-Pokemon team with type coverage analysis, defensive weakness tracking, and stat summaries
- **Type Chart** -- Interactive type effectiveness grid with quick-lookup mode
- **Who's That Pokemon?** -- Classic silhouette guessing game with difficulty settings
- **Evolution Explorer** -- Visual evolution chains with conditions (level, stone, trade)

## Setup

```bash
npm install
npm run dev
```

## Tech Stack

- React 19 + Vite
- React Router v6
- TanStack Query with localStorage persistence
- Tailwind CSS v4
- PokeAPI v2
